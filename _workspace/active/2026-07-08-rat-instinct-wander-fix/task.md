# 작업 패킷: 쥐 숙주 본능 이동 왕복 문제 수정

## 목표

쥐 숙주가 랜덤/생물적 배회처럼 움직이지 않고 위아래로 반복 왕복하는 문제를 수정한다.

## 사용자 제보

- 현재 쥐가 랜덤으로 움직이는지 질문.
- 실제로는 위아래로 계속 왕복하고 있음.
- 좌우 조종 시 대각선으로 움직임.
- 낮은 바이러스 조종 능력 때문이면 가능하지만, 그렇지 않거나 의도된 것이 아니라면 버그로 봐야 함.

## 원인 판단

- 현재 `RatHostController`의 `hostInstinctMoveDirection` 기본값은 `Vector3.forward`.
- `UpdateHostInstinctDirection()`은 경계에서 축만 반전하므로 위아래 왕복이 발생한다.
- `RatHostControlModel`의 낮은 조종력 입력 혼합은 의도된 설계다.
- 단, 본능 이동이 고정 왕복인 것은 쥐 생물 행동/배회 의도에 맞지 않는다.

## 구현 범위

- 숙주 본능 이동에 초기 방향 랜덤화와 주기적 방향 전환 추가
- 경계에 닿을 때 단순 반전이 아니라 벽을 타는 방향으로 회피
- 낮은 조종력 대각 입력과 높은 조종력 직진 입력의 기존 테스트 유지
- EditMode 테스트 추가

## 비범위

- 네비게이션/AI 경로 탐색 도입
- 장애물 회피 완성형 AI
- 조종력/변이 수치 대규모 밸런스 조정
- 기본 내부 미니게임 선택 방식 변경
- `ProjectSettings.asset` 기존 변경

## 수용 기준

1. 본능 이동은 시작부터 항상 위쪽 고정 방향만 사용하지 않는다.
2. 경계에 닿으면 위아래 반전만 반복하지 않고 좌우 또는 벽을 타는 방향으로 회피한다.
3. 낮은 조종력에서는 입력과 본능이 섞여 대각 이동할 수 있다.
4. 높은 조종력에서는 입력 방향이 본능보다 우선한다.
5. 컴파일 검증과 관련 테스트 컴파일이 통과한다.
