# 인계 메모

## 현재 상태

- 작업 상태: 빌드 실행본 성공 루프 최종 확인 완료 / 사용자 체감 확인 별도
- QA 판정: 빌드 성공 루프 최종 확인 통과, 제한 조건 명시
- 총괄 관리자 판정: 자동/대리 입력 기준 성공 루프 차단 해소
- 코드/씬/ProjectSettings 변경: 없음

## 확인된 내용

- Editor Play 전체 성공 루프 통과.
- Editor Play 실패 복귀 루프 통과.
- Windows 빌드 실행본 네 기준 해상도 구동 확인.
- 빌드 실행본 Player.log에 크래시/예외 패턴 없음.
- 16:9 해상도별 RatHost 기본 화면에서 HUD/카메라 핵심 가림 문제 없음.
- 1280x720 빌드 실행본에서 F6 내부 미니게임 진입, 실패 패널, Space 복귀 확인.
- 1280x720 빌드 실행본에서 F6 `면역 신호 억제` 성공, 변이 선택, `잠복 강화` 선택, RatHost 복귀 확인.

## 아직 닫지 않은 항목

- 자연 경계도 100% 도달 후 기본 `백혈구 회피`에서 조각 3개를 직접 수집하는 빌드 실행본 루프는 미검증.
- 사용자 조작감과 난이도 체감은 미검증.

## 다음에 할 일

1. 사용자 수동 플레이로 조작감, 난이도, 설명 없이 목표 이해 여부를 확인한다.
2. 필요하면 자연 경계도 100%와 기본 `백혈구 회피` 조각 수집 루프를 빌드 실행본에서 별도 수동 검증한다.
3. 커밋 요청 시 문서와 artifacts만 선별하고 ProjectSettings diff가 없는지 다시 확인한다.

## 주의

- `UnityProject/ProjectSettings/ProjectSettings.asset`에는 변경을 남기지 않는다.
- F6 `면역 신호 억제` 성공 루프와 자연 경계도/백혈구 회피 성공 루프를 혼동해 기록하지 않는다.
- 빌드 wrapper의 Sentis/Inference shader warning은 기존과 같은 패키지 경고로 기록하되, 별도 해결 작업으로 분리한다.
