# 작업 로그

## 2026-07-02

- 기존 공식 기획 문서에서 면역 경계도가 숙주 조종 모드와 내부 바이러스 모드를 연결하는 핵심 게이지임을 확인했다.
- 현재 프로토타입 구현 기준에서 기본 시간 상승은 `0`, 소음 배관 상호작용은 `+15` 경계도 상승으로 동작하는 것을 확인했다.
- 실제 면역 반응 참고 자료를 확인해 선천면역, 장벽, 바이러스 패턴 인식, 인터페론/사이토카인, 염증, 면역 세포 동원을 게임 규칙으로 추상화했다.
- 상세 문서를 시스템, 상호작용, UI 피드백으로 분리해 작성했다.
- 새 문서가 다음 작업에서 참조될 수 있도록 README와 `docs/agents/agent-reference-map.md`를 갱신했다.
- 사용자 피드백을 반영해 바이러스 패턴 노출 트리거, 오염 구역 적용 조건, 숙주 본능과 강제 조종, 내부 미니게임 다양화, 반복 내부 대응 난이도 기준을 보강했다.
- 숙주 본능/강제 조종 기준 문서와 내부 면역 대응 미니게임 유형 문서를 추가했다.
