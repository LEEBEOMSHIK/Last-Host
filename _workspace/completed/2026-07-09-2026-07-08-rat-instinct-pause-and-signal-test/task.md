# 작업 패킷: 쥐 본능 정지 리듬과 면역 신호 억제 테스트 진입

## 목표

쥐 숙주 자동 배회가 계속 걷기만 하는 문제를 줄이고, 조종력 보상 후 강제 조종 경계도 처리 기준을 확인하며, 면역 신호 억제 미니게임을 사용자가 바로 테스트할 수 있게 한다.

## 사용자 요청

1. 자동 움직임이 방향을 바꾸긴 하지만 멈추거나 쥐처럼 움직이지 않고 계속 돌아다닌다.
2. 미니게임 보상으로 조종력을 얻은 뒤 조종 관련 경계도 증가가 없어지는 것이 정상인지 확인한다.
3. 추가된 면역 신호 억제 미니게임을 테스트할 수 있도록 준비한다.

## 구현 범위

- 쥐 본능 이동에 이동/짧은 정지 리듬 추가
- 본능 정지 중 입력이 없으면 자동 이동을 멈춤
- 본능 정지 중에도 플레이어 입력은 처리
- 높은 조종력에서는 본능과 반대 입력도 강제 조종 디메리트가 발생하지 않는 테스트 추가
- 개발/테스트용 `F6` 키로 면역 신호 억제 미니게임 진입

## 비범위

- 완성형 동물 AI/경로 탐색
- 기본 내부 미니게임 랜덤/원인별 자동 선택
- 면역 신호 억제 전용 시각 노트 UI 제작
- `ProjectSettings.asset` 기존 define 변경

## 수용 기준

1. 입력이 없을 때 쥐가 계속 걷기만 하지 않고 짧게 멈출 수 있다.
2. 낮은 조종력의 대각 입력/강제 조종 디메리트는 유지된다.
3. `신경 조종` 수준의 높은 조종력에서는 본능 반대 입력도 강제 조종 디메리트가 발생하지 않는다.
4. 기본 면역 경계도 100% 진입은 백혈구 회피로 유지된다.
5. Play 중 `F6`으로 면역 신호 억제 미니게임에 들어갈 수 있다.
