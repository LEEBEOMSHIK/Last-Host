## 해야 할 일

1. `RatRiskInteractable`의 Space 입력 경로를 공통 입력 래퍼로 통일한다.
2. 기본 시간 경과 면역 상승을 현재 프로토타입에서는 0으로 조정한다.
3. 무행동 시간 경과, 위험 이벤트, 상호작용 입력 매핑에 대한 테스트를 보정한다.
4. 구현 주체와 분리된 검증을 수행한다.
5. 프로젝트 총괄 관리자 판정을 기록한다.
