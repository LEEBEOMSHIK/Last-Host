# 작업 패킷: 면역 신호 억제 시각 미니게임 1차 구현

## 목표

`면역 신호 억제`가 텍스트 상태 모델이 아니라 플레이 중 눈으로 확인 가능한 리듬형 미니게임으로 보이도록 1차 시각화를 붙인다.

## 사용자 요청

면역 신호 억제 미니게임이 실제로 구현된 것으로 느껴지도록 다음 작업을 진행한다.

## 구현 범위

- `F6` 진입 후 전용 신호 억제 패널 표시
- 신호 트랙, 판정선, 신호 마커 표시
- 다음 신호까지 남은 시간에 따라 마커가 판정선으로 접근
- 판정 가능 타이밍과 입력 결과를 텍스트/색으로 피드백
- 성공/실패 루프는 기존 `ImmuneSignalSuppressionModel`과 세션 전환 사용

## 비범위

- 사운드 싱크
- 2키/가짜 신호/롱노트
- 원인별 자동 미니게임 선택
- 신규 아트 에셋 생성
- 전체 내부 미니게임 UI 리디자인
- `ProjectSettings.asset` 기존 define 변경

## 수용 기준

1. 쥐 숙주 모드에서 `F6`으로 면역 신호 억제에 들어가면 전용 패널이 보인다.
2. 패널에는 신호 트랙, 판정선, 움직이는 신호 마커가 있다.
3. 신호가 판정선에 가까워지면 `지금 차단` 상태를 확인할 수 있다.
4. `Space` 정확 입력 시 신호 차단 수가 증가하고, 실패 입력은 경보 증폭/안정도 감소로 이어진다.
5. 백혈구 회피 기본 미니게임은 기존 동작을 유지한다.
