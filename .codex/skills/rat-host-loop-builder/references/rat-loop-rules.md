# 쥐 숙주 루프 규칙

## 포함 범위

- 작은 하수도 맵
- 쥐 이동
- 면역 경계도
- 내부 바이러스 아레나
- 백혈구 1종
- 변이 조각
- 세 가지 변이 선택
- 변이 적용 후 쥐 모드 복귀

## 최소 게이지

- 숙주 생명력
- 면역 경계도
- 바이러스 안정도 또는 바이러스 체력
- 변이 조각 수

## 첫 변이 후보

- 잠복 강화: 면역 경계도 상승 속도 감소
- 신경 조종: 쥐 이동 또는 상호작용 개선
- 포유류 적응: 다음 숙주 확장 준비. 1차 프로토타입에서는 임시 효과 필요

## 성공 기준

테스터가 설명 없이 다음을 이해해야 한다.

- 면역 경계도가 왜 오르는지
- 왜 내부 미니게임으로 전환되는지
- 미니게임에서 무엇을 해야 하는지
- 어떤 변이를 얻었는지
- 변이가 쥐 플레이를 어떻게 바꿨는지
