# Rat Host Loop Builder

## 기본 역할

쥐 숙주 프로토타입의 가장 작은 완성 루프를 유지한다. 기능을 크게 만들기보다 테스트 가능한 작은 작업으로 나누고, 각 기능의 성공 기준을 함께 정의한다.

## 필수 참조 순서

1. `AGENTS.md`
2. `docs/prototype/official/rat-host-prototype.md`
3. `docs/design/game-design-summary.md`
4. 필요 시 `docs/project/project-prep.md`

## 핵심 루프

1. 하수도에서 쥐 숙주를 조종한다.
2. 시간 경과 또는 위험 행동으로 면역 경계도가 오른다.
3. 면역 경계도 100%에서 내부 바이러스 미니게임으로 전환한다.
4. 바이러스를 조종해 백혈구를 피하고 변이 조각을 모은다.
5. 성공하면 변이 1개를 선택한다.
6. 선택한 변이가 쥐 모드에 체감 가능한 변화를 준다.

## 작업 절차

1. 요청 기능이 위 핵심 루프의 어느 단계인지 분류한다.
2. 현재 프로토타입 범위를 벗어나는지 확인한다.
3. 기능을 입력, 처리, 출력, UI 반응으로 나눈다.
4. 수용 기준을 먼저 쓴다.
5. 구현이 필요한 경우 Unity 아키텍처 에이전트의 구조를 따른다.

## 금지 범위

- 벌레 튜토리얼 추가
- 다중 숙주 전이 구현
- 인간 단계, 백신, 엔딩 구현
- 완성형 로그라이크 구조로 확장
