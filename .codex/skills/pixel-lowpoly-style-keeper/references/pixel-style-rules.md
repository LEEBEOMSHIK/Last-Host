# 도트풍 하이브리드 2.5D 스타일 규칙

## 핵심 표현

- 3D 환경·게임플레이 공간·충돌·카메라를 사용한다.
- 캐릭터는 저폴리 3D 원본을 제작 기준으로 두고, 쥐 프로토타입 런타임에서는 8방향 프리렌더드 스프라이트로 표시한다.
- 환경 표면과 캐릭터 스프라이트의 렌더링 결과가 같은 도트풍으로 보이게 한다.
- 캐릭터 실루엣은 단순하고 읽기 쉬워야 한다.
- 바이러스는 귀엽지만 기묘해야 한다.

## 렌더링 기준

- 저해상도 렌더 타겟 또는 픽셀 후처리를 고려한다.
- 텍스처 필터링은 픽셀 느낌을 해치지 않도록 설정한다.
- 색상 수를 과도하게 늘리지 않는다.
- 조명은 형태를 읽기 쉽게 단순화한다.
- 캐릭터의 구운 명암과 3D 환경 조명이 충돌하지 않게 한다.
- 8방향 프레임은 같은 캔버스·피벗·화면 점유 크기를 유지한다.

## 프로토타입 기준

- 최종 아트보다 조작감과 루프 검증을 우선한다.
- 플레이스홀더는 의도된 저폴리 스타일과 구분되게 문서화한다.
- 현재 3D 프리미티브 캐릭터는 실제 스프라이트 통합 전까지 게임플레이 플레이스홀더로 사용할 수 있다.
- 도트풍이 단순 저품질처럼 보이지 않도록 카메라와 렌더링 기준을 먼저 잡는다.
