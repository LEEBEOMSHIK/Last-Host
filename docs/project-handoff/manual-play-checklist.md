# 사용자 수동 플레이 체크리스트

최종 갱신: 2026-07-10

## 상태

보류. 자동/대리 입력 기준의 핵심 루프 검증은 완료됐지만, 아래 항목은 사용자가 직접 플레이해야 판단할 수 있다.

## 실행 대상

- Windows 빌드 실행본: `Builds/RatHostPrototype/LastHostPrototype.exe`
- 권장 해상도: 창 모드 `1280x720` 이상

## 체크 항목

| 항목 | 확인 | 메모 |
| --- | --- | --- |
| `WASD` 이동이 답답하지 않고 의도한 방향으로 움직인다 |  |  |
| 하수도에서 상호작용 가능한 배관/밸브 대상을 눈으로 찾을 수 있다 |  |  |
| 근접 시 `소음 배관 조사 가능` HUD 문구가 이해된다 |  |  |
| `Space` 입력 후 면역 경계도가 오른다는 점이 이해된다 |  |  |
| 면역 경계도 100% 또는 `F6` 진입 후 내부 미니게임 목표를 파악할 수 있다 |  |  |
| 변이 선택 화면에서 세 선택지의 차이를 대략 이해할 수 있다 |  |  |
| 변이 선택 후 쥐 숙주로 돌아온 것을 알아볼 수 있다 |  |  |
| 전체 흐름이 설명 없이도 최소한 따라갈 수 있다 |  |  |

## 판정 기준

- `통과`: 위 항목 대부분이 별도 설명 없이 이해된다.
- `보류`: 일부는 가능하지만 목표, 입력, 변화가 헷갈린다.
- `실패`: 이동 외 행동, 내부 미니게임 목표, 변이 적용 중 하나 이상이 거의 이해되지 않는다.

## 기록 방법

플레이 후 이 파일의 `확인`과 `메모` 칸을 채우거나, 채팅으로 항목별 결과만 알려주면 작업 기록에 반영한다.
