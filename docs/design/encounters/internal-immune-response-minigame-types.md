# 내부 면역 대응 미니게임 유형

## 목적

내부 미니게임은 면역 경계도 100%에 도달했을 때 바이러스가 숙주 내부 압박에 대응하는 장면이다. 이 장면은 무조건 백혈구와 싸우는 전투로 고정하지 않는다. 경계도가 어떤 이유로 올랐는지, 바이러스가 어떤 변이를 선택했는지, 숙주 상태가 어떤지에 따라 다른 미니게임을 사용할 수 있다.

## 현재 프로토타입 위치

1차 쥐 숙주 프로토타입은 최소 루프 검증을 위해 다음 형태를 우선한다.

- 바이러스 아바타 이동
- 백혈구 1종 회피
- 변이 조각 수집
- 성공 후 변이 선택

이것은 첫 검증용 대표 유형일 뿐, 내부 미니게임 전체 방향을 백혈구 전투로 확정한다는 뜻은 아니다.

## 성공과 실패 결과

내부 미니게임은 보상을 얻기 위한 안전한 방이 아니다. 면역 경계도 100%는 숙주의 면역계가 바이러스를 강하게 인지해 내부 대응을 시작했다는 뜻이며, 미니게임 진입 자체가 면역계에 대응 경험을 남긴다.

1차 프로토타입의 결과 방향은 다음처럼 둔다.

| 결과 | 보상 | 면역계 변화 | 바이러스 피해 |
| --- | --- | --- | --- |
| 성공 | 변이 선택 가능 | 대응 경험 누적, 다음 대응 소폭 강화 | 없음 또는 최소 |
| 실패 | 변이 보상 없음 또는 제한 | 대응 경험 추가 누적, 다음 대응 더 강화 | 다음 바이러스 안정도 감소 |

실패 시 수집한 변이 조각은 기본적으로 폐기한다. 이후 필요하면 `일부 조각 유지`, `약한 보상`, `조건부 보상` 같은 완화 장치를 별도 설계할 수 있지만, 첫 기준은 실패 보상을 제한하는 쪽으로 둔다.

바이러스 피해는 우선 `다음 내부 미니게임 시작 안정도 감소`로 해석한다. 이 방식은 즉시 게임오버보다 가볍지만, 반복 실패가 누적되면 다음 대응이 더 위험해진다는 압박을 만든다. 숙주 생명력 직접 피해, 숙주 유지 실패, 감염력 손상은 이후 마일스톤에서 확장 후보로 둔다.

## 미니게임 유형 후보

| 유형 | 플레이 핵심 | 어울리는 경계도 원인 | 실패 압박 |
| --- | --- | --- | --- |
| 면역 회피 | 백혈구, 항체, 감시 패턴을 피함 | 면역 포착, 염증 동원 | 바이러스 안정도 감소 |
| 신호 억제 | 경보 노드나 사이토카인 신호를 끊음 | 인터페론/사이토카인 경보 | 다음 경계도 상승량 증가 |
| 잠복 유지 | 움직임을 줄이고 안전 지대에서 버팀 | 바이러스 패턴 노출 | 제한 시간 동안 포착 회피 |
| 세포 위장 | 숙주 세포 패턴에 맞춰 통과함 | 감염 세포 이상 신호 | 잘못된 패턴 선택 시 포착 |
| 손상 복구 | 숙주 손상 지점을 안정화함 | 조직 자극, 오염 노출 | 숙주 생명력 추가 손상 |
| 변이 회수 | 압박 속에서 변이 조각을 모음 | 모든 원인 공통 | 조각 부족 시 약한 보상 |

## 경계도 원인과 미니게임 연결

| 경계도 주원인 | 권장 미니게임 |
| --- | --- |
| 오염 구역 체류, 독성 접촉 | 손상 복구, 면역 회피 |
| 바이러스 흔적 노출 | 잠복 유지, 신호 억제 |
| 강제 숙주 조종 | 세포 위장, 신호 억제 |
| 면역 포착 반복 | 면역 회피 |
| 장기 체류 | 면역 회피와 신호 억제 복합 |

## 반복 플레이 난이도

현재 구조에서는 면역 경계도 100%에 도달해야 내부 미니게임이 시작된다. 따라서 미니게임 난이도는 `경계도 수치가 차는 중`에 바로 올라가는 것이 아니라, `내부 대응을 여러 번 겪은 뒤 다음 내부 대응이 더 정교해지는 방식`이 자연스럽다.

반복 적용 후보:

- 두 번째 내부 대응부터 감시 패턴 1개 추가
- 같은 원인으로 재진입하면 해당 원인에 맞는 장애물 추가
- 잠복/공존 변이를 선택하면 다음 강화폭 감소
- 강한 침식/증식 변이를 선택하면 보상은 크지만 다음 대응 압박 증가
- 실패한 내부 대응은 성공보다 큰 대응 경험을 남겨 다음 내부 미니게임 난이도를 더 올린다.

첫 구현 후보는 단순하게 `면역 대응 경험` 카운트를 사용한다. 내부 미니게임에 진입하면 경험이 오르고, 실패하면 추가 경험을 더한다. 경험이 높을수록 백혈구 속도, 수, 등장 간격, 추적 적극성 중 하나 이상을 단계적으로 강화한다.

## 설계 원칙

- 내부 미니게임은 벌칙이 아니라 변이와 적응을 얻는 압박 장면이다.
- 백혈구는 대표 적일 수 있지만 항상 중심 적일 필요는 없다.
- 원인 없는 난이도 상승보다, 플레이어가 누적 원인을 이해할 수 있는 대응 변화를 우선한다.
- 성공해도 면역계는 바이러스를 조금 더 학습하고, 실패하면 보상 제한과 바이러스 피해를 함께 남긴다.
- 첫 프로토타입은 단순한 회피/수집으로 검증하고, 이후 원인별 미니게임을 추가한다.
