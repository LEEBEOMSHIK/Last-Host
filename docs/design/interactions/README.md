# 상호작용 기획

플레이어가 접근해 입력하거나 접촉해 결과를 만드는 오브젝트와 행동을 정리한다.

## 포함 문서

- 소음 배관 같은 조사 가능 오브젝트
- 문, 통로, 막힌 경로
- 먹이, 감염 기회, 전이 대상
- 위험 행동과 보상/페널티

## 작성 기준

- 오브젝트가 무엇인지
- 플레이어가 어떤 조건에서 상호작용할 수 있는지
- 입력 후 어떤 값이 변하는지
- 왜 그 결과가 게임적으로 타당한지
- HUD나 시각 피드백이 무엇인지
