# 숙주 본능과 강제 조종

## 목적

숙주는 단순한 플레이어 아바타가 아니라 원래의 본능, 회피 성향, AI 행동 패턴을 가진 생물이다. 바이러스가 숙주를 조종할 때는 이 본능을 부드럽게 이용할 수도 있고, 억지로 거스를 수도 있다. 이 문서는 강제 조종이 면역 경계도와 어떻게 연결되는지 정의한다.

## 기본 원칙

- 플레이어가 숙주를 조작한다는 이유만으로 면역 경계도를 올리지 않는다.
- 숙주의 본능과 맞는 행동은 낮은 비용으로 허용한다.
- 숙주의 본능과 충돌하는 행동을 반복하면 `조종 부하`가 쌓인다.
- 조종 부하는 면역 경계도 상승, 숙주 반응 지연, 짧은 비틀거림 같은 결과로 표현할 수 있다.

## 쥐 숙주 기준

쥐는 작은 포유류 숙주로서 다음 본능을 가진 것으로 본다.

| 본능 | 자연스러운 행동 | 강제 조종으로 볼 수 있는 행동 |
| --- | --- | --- |
| 어두운 곳 선호 | 벽, 그늘, 좁은 길을 따라 이동 | 밝고 열린 공간으로 계속 이동 |
| 소음 회피 | 큰 소리나 진동에서 멀어짐 | 소음 배관, 진동원 쪽으로 반복 접근 |
| 오염/통증 회피 | 독성 물, 날카로운 구조물 회피 | 오염 구역에 오래 머무르거나 강제 통과 |
| 위험 회피 | 위협 방향에서 도망감 | 공포 상태에서 위협 쪽으로 역주행 |
| 에너지 보존 | 짧은 이동과 탐색 반복 | 무리한 돌진, 급회전, 강제 상호작용 반복 |

## 조종 단계

| 단계 | 의미 | 경계도 처리 |
| --- | --- | --- |
| 자연 유도 | 숙주 본능과 맞는 방향으로 이동 | 상승 없음 또는 매우 낮음 |
| 약한 개입 | 본능과 약간 다르지만 위험하지 않음 | 상승 없음 또는 소량 |
| 강제 조종 | 본능과 반대되는 위험 행동 | 조종 흔적 상승 |
| 과부하 | 강제 조종을 짧은 시간에 반복 | 경계도 상승과 숙주 반응 저하 후보 |

## 게임 적용 후보

| 상황 | 처리 후보 | 피드백 예시 |
| --- | --- | --- |
| 소음 배관을 조사함 | 위험 상호작용으로 경계도 상승 | `소음/조직 자극 +15` |
| 공포 상태에서 위협 쪽으로 계속 이동 | 조종 부하 누적 | `강제 조종 +8` |
| 독성 물웅덩이에 오래 머무름 | 오염 자극과 조종 부하 동시 후보 | `오염 자극 +10` |
| 급회전/돌진을 짧게 반복 | 숙주 이상 행동으로 처리 | `조종 흔적 +6` |
| 잠복 강화 변이 보유 | 같은 강제 조종의 상승량 감소 | `잠복으로 상승 억제` |

## 구현 전 결정할 사항

이 문서는 구현 기준이 아니라 설계 기준이다. 실제 구현 전에는 다음을 별도 작업으로 정한다.

- 쥐의 기본 AI/본능 상태를 어느 정도까지 시뮬레이션할지
- 플레이어 입력과 AI 본능이 충돌할 때 입력을 얼마나 늦추거나 제한할지
- 조종 부하를 별도 게이지로 둘지, 면역 경계도 상승 원인으로만 둘지
- 강제 조종이 재미를 해치지 않도록 어떤 피드백과 보상을 붙일지
