# 면역 경계도 시스템

## 목적

`면역 경계도`는 숙주 몸이 바이러스의 존재와 이상 행동을 얼마나 강하게 알아차리고 있는지를 나타내는 0-100 누적 압박값이다. 이 값은 실제 면역학을 그대로 재현하는 의학 시뮬레이션이 아니라, `마지막 숙주`의 핵심 루프를 플레이어가 이해하기 쉬운 게임 규칙으로 압축한 값이다.

1차 쥐 숙주 프로토타입에서는 면역 경계도가 100%에 도달하면 숙주 조종 모드에서 내부 바이러스 미니게임으로 전환한다. 즉, 경계도는 단순한 체력이나 위험도 UI가 아니라 바깥 탐험과 내부 면역 대응을 연결하는 전환 장치다.

## 핵심 정의

- `0%`: 숙주 내부에서 바이러스가 아직 크게 드러나지 않은 상태
- `1-99%`: 바이러스 활동, 숙주 손상, 위험 행동 때문에 면역계가 점점 활성화되는 상태
- `100%`: 국소 면역 반응이 충분히 커져 내부 바이러스 미니게임으로 전환되는 상태

면역 경계도는 다음 값과 구분한다.

- `숙주 생명력`: 숙주 몸 자체의 손상 정도
- `바이러스 안정도`: 내부 미니게임에서 바이러스가 버틸 수 있는 정도
- `침식도`: 바이러스가 숙주를 얼마나 깊게 조종하거나 장악했는지
- `인간 위험도`: 인간 사회나 외부 관찰자가 바이러스 사건을 인지하는 정도

## 실제 면역 반응에서 빌리는 축

### 1. 장벽 돌파와 조직 자극

실제 숙주는 피부, 점막, 점액, 산, 효소 같은 선천 방어 장벽으로 외부 침입을 먼저 막는다. 게임에서는 이 개념을 `숙주 몸이 평소 상태에서 벗어나 자극을 받는 순간`으로 바꾼다.

게임에서 이것은 가장 직관적으로 `더럽고 위험한 곳에 들어가면 면역 경계도가 오른다`는 느낌으로 적용할 수 있다. 다만 모든 더러운 배경이 자동으로 경계도를 올리는 것은 아니다. 플레이어가 오염원에 접촉하거나, 위험 구역에 오래 머물거나, 상처/피로 상태에서 더러운 구역을 통과할 때 경계도가 오른다.

게임 적용 예:

- 더러운 물, 독성 물웅덩이, 오염된 쓰레기 더미 접촉
- 하수도 진흙, 오물, 부패한 먹이 주변에 오래 머무름
- 거친 배관이나 좁은 틈을 무리하게 통과
- 쥐가 몸을 비틀어 좁은 틈이나 막힌 통로를 강제로 통과
- 상처, 점막 자극, 오염 노출을 암시하는 환경 접촉

### 2. 바이러스 패턴 노출

실제 선천면역은 바이러스 RNA 같은 비정상 분자 패턴을 감지해 초기 반응을 시작한다. 게임에서는 이를 `바이러스가 너무 활발히 움직이거나 증식해 숙주 내부 감시망에 흔적을 남기는 상황`으로 추상화한다.

이 축은 바깥 이동만으로 발동하지 않는다. 플레이어가 바이러스다운 행동을 강하게 하거나, 내부 세포에 흔적을 남기거나, 은신보다 확산을 우선할 때 적용한다.

게임 적용 예:

- 감염 지점을 빠르게 넓혀 `바이러스 흔적`을 남김
- 내부 세포 침투, 복제, 변이 조각 흡수를 짧은 시간 안에 반복함
- 은신 변이가 약한 상태에서 강한 바이러스 능력을 사용함
- 내부 미니게임에서 방어 감시망이나 면역 세포에게 자주 포착됨
- 변이 선택으로 `증식/침식` 계열을 강화해 강한 효과를 얻는 대신 흔적을 크게 남김

적용 규칙 후보:

| 트리거 | 상승 의미 | 피드백 예시 |
| --- | --- | --- |
| 내부 세포 침투 반복 | 바이러스 흔적 누적 | `바이러스 흔적 +8` |
| 강한 변이 능력 사용 | 은신보다 효과를 우선 | `변이 흔적 +12` |
| 빠른 감염 확장 | 감시망에 노출 | `감염 흔적 +10` |
| 내부 감시망 접촉 | 패턴 인식에 걸림 | `면역 포착 +15` |

현재 쥐 숙주 프로토타입 구현값:

| 트리거 | 현재 효과 | UI 피드백 |
| --- | --- | --- |
| 내부 미니게임 중 백혈구 접촉 | 바이러스 패턴 노출 누적 `+8` | `면역 포착 +8` |
| 포착 흔적을 가진 채 변이 선택 성공 | 쥐 숙주 복귀 경계도에 누적 포착값 가산 | `면역 포착 흔적 +8` |
| 내부 미니게임 실패 후 복귀 | 기존 실패 복귀 경계도 60% 유지, 포착 흔적 초기화 | 실패 UI 후 복귀 |

### 3. 감염 세포의 이상 신호

실제 면역계는 바이러스에 감염된 세포나 비정상 표면을 보이는 세포를 탐지할 수 있다. 게임에서는 이를 `숙주 내부에서 바이러스 조종 흔적이 겉으로 새어 나오는 것`으로 바꾼다.

숙주에게는 원래의 본능 또는 AI 행동 패턴이 있다. 바이러스가 그 흐름을 부드럽게 유도하면 경계도 상승이 낮고, 숙주 본능과 반대되는 행동을 강제로 시키면 `조종 부하`가 쌓여 면역 경계도가 오른다.

게임 적용 예:

- 쥐가 원래 피하려는 밝은 공간, 소음, 오염원 쪽으로 억지로 이동함
- 공포/회피 AI가 발동한 상태에서 플레이어가 반대 방향으로 계속 입력함
- 신경 조종으로 급회전, 무리한 돌진, 강제 상호작용을 반복함
- 숙주 생명력이 낮거나 피로한 상태에서 추가 조작을 강행함

이 규칙은 단순히 `플레이어가 조작하면 벌을 받는다`가 아니다. 숙주의 자연스러운 행동 범위 안에서 움직이면 낮은 비용으로 조종하고, 본능을 거스르는 강제 조작은 강한 효과와 경계도 상승을 함께 가진다. 상세 기준은 `../hosts/host-instinct-control.md`에 둔다.

### 4. 인터페론/사이토카인 경보

실제 감염 초기에는 세포와 면역 세포가 화학 신호를 통해 주변 조직에 위험을 알리고 면역 반응을 증폭한다. 게임에서는 이를 `한 번 오른 경계도가 주변 면역 압박으로 이어지는 누적 경보`로 표현한다.

게임 적용 예:

- 경계도 구간이 올라갈수록 HUD 경고가 강해짐
- 특정 변이가 없으면 같은 위험 행동의 경계도 상승량이 커짐
- 내부 미니게임을 여러 번 겪을수록 다음 내부 대응의 압박이 강해짐

현재 구조에서는 면역 경계도 100%에 도달해야 내부 미니게임이 시작된다. 따라서 `경계도가 오르는 중에 미니게임 난이도가 바로 오른다`로 해석하지 않는다. 더 적절한 해석은 `내부 대응을 반복할수록 숙주 면역계가 같은 바이러스에 더 빠르게 반응해 다음 미니게임이 어려워진다`이다.

반복 내부 대응 적용 후보:

| 조건 | 다음 내부 미니게임 변화 |
| --- | --- |
| 내부 미니게임 1회 완료 | 기본 압박 유지 |
| 내부 미니게임 2회 이상 완료 | 감시 패턴 추가 또는 반응 속도 소폭 증가 |
| 같은 위험 행동을 반복해 재진입 | 해당 원인과 연결된 패턴 등장 |
| 잠복/공존 계열 변이 선택 | 다음 내부 대응 강화폭 감소 |

### 면역 대응 경험

`면역 대응 경험`은 숙주의 면역계가 같은 바이러스와 내부에서 맞붙은 기록이다. 플레이어가 면역 경계도를 일부러 올려 내부 미니게임에 진입할 수는 있지만, 그 행동은 공짜 보상 진입이 아니라 면역계에 정보를 남기는 위험한 선택이어야 한다.

1차 프로토타입에서의 해석은 다음과 같다.

| 사건 | 면역계 해석 | 게임 결과 |
| --- | --- | --- |
| 내부 미니게임 진입 | 면역계가 바이러스 대응을 시작함 | 면역 대응 경험이 오른다 |
| 내부 미니게임 성공 | 바이러스가 적응했지만 면역계도 일부 학습함 | 변이 보상 가능, 다음 대응 강화 |
| 내부 미니게임 실패 | 면역계가 바이러스를 더 강하게 포착함 | 보상 제한, 추가 대응 경험, 바이러스 피해 |

기본 방향은 `성공 = 변이 획득, 면역계 학습`이고, `실패 = 변이 없음, 면역계 추가 학습, 바이러스 손상`이다. 이 규칙 때문에 플레이어는 경계도 상승을 전략적으로 유도할 수 있지만, 반복 진입과 실패는 다음 내부 대응을 더 어렵게 만든다.

면역 대응 경험은 다음 내부 미니게임의 백혈구 수, 이동 속도, 등장 간격, 추적 적극성, 감시 패턴 같은 난이도 요소에 연결한다. 정확한 수치는 구현 단계에서 정하되, 문서 기준으로는 `경험이 쌓일수록 다음 대응이 더 정교해진다`를 원칙으로 둔다.

### 5. 염증과 면역 세포 동원

실제 염증은 감염 지점에 면역 세포와 단백질을 불러와 침입자를 막고 확산을 제한한다. 게임에서는 이를 `내부 바이러스 미니게임의 시작 조건`으로 사용한다.

게임 적용 예:

- 100% 도달 시 내부 바이러스 대응 미니게임으로 전환
- 백혈구 회피, 신호 억제, 잠복 유지, 변이 조각 회수 등 다양한 미니게임으로 압박을 표현
- 미니게임 성공 시 변이 선택으로 압박에 적응

1차 프로토타입은 백혈구 회피와 변이 조각 수집으로 최소 루프를 검증한다. 장기적으로는 내부 미니게임을 모두 전투로 만들지 않고, 숙주 상태와 경계도 원인에 맞는 여러 유형으로 분리한다. 상세 후보는 `../encounters/internal-immune-response-minigame-types.md`에 둔다.

### 6. 적응면역과 기억

실제 적응면역은 느리지만 더 정확하고, 이후 같은 침입자에 더 빠르게 반응할 수 있다. 1차 쥐 프로토타입에서는 이 축을 직접 구현하지 않는다. 장기 설계에서는 같은 숙주에 오래 머물수록 경계도 기준이 불리해지거나, 백신/인간 의료 시스템과 연결하는 후보로 남긴다.

## 게임에서 경계도를 올리는 원인

면역 경계도 상승 원인은 플레이어가 이해할 수 있는 행동 단위로 제시해야 한다.

| 원인 범주 | 실제 반응에서 빌린 개념 | 게임 내 의미 | 예시 |
| --- | --- | --- | --- |
| 감염 흔적 노출 | 바이러스 패턴 인식 | 바이러스 활동이 너무 드러남 | 내부 세포 침투 반복, 빠른 증식, 강한 변이 능력 |
| 조직 자극 | 장벽 손상, 염증 시작 | 숙주 몸이 물리적으로 자극받음 | 오염 구역 체류, 독성 물, 거친 배관, 상처 |
| 강제 숙주 조종 | 감염 세포 이상 신호 | 숙주 본능과 다른 행동을 억지로 시킴 | 공포 상태 역주행, 급회전 반복, 강제 상호작용 |
| 위험 환경 접촉 | 방어 장벽 자극 | 오염과 손상 위험이 커짐 | 하수도 오물, 날카로운 구조물, 부패한 먹이 |
| 면역 포착 | 면역 감시망 반응 | 내부 감시망에 걸림 | 미니게임 중 감시 패턴 접촉, 면역 세포 접촉 |
| 장기 체류 | 감염 지속과 면역 적응 | 같은 숙주에 오래 머묾 | 이후 마일스톤 후보 |

## 경계도를 낮추거나 억제하는 원인

면역 경계도는 항상 오르기만 하는 값으로 고정하지 않는다. 장기적으로는 플레이어의 선택에 따라 `빠른 침식`과 `조용한 공존`이 갈라져야 한다.

| 억제 방식 | 게임 의미 | 연결 후보 |
| --- | --- | --- |
| 잠복 강화 | 바이러스 흔적을 덜 남김 | 변이 선택지 |
| 조심스러운 이동 | 숙주 손상을 줄임 | 쥐 조작, 환경 회피 |
| 공존 계열 선택 | 숙주를 덜 망가뜨림 | 공존도, 장기 생존 |
| 면역 교란 변이 | 같은 행동의 상승량 감소 | 변이/성장 시스템 |

## 현재 쥐 프로토타입 규칙

현재 프로토타입은 플레이어가 원인을 읽기 전까지 경계도가 자동으로 오르는 상황을 피하기 위해 대기 시간에 따른 기본 상승을 꺼 둔다.

| 항목 | 현재 기준 | 의미 |
| --- | --- | --- |
| 최대치 | `100%` | 도달 시 내부 바이러스 모드 전환 |
| 기본 시간 상승 | `0` | 가만히 있어도 오르는 구조는 현재 비활성 |
| 위험 행동 기본 상승량 | `20` | 위험 행동의 기본 경계도 보너스 |
| 소음 배관 위험도 | `0.75` | 기본 상승량의 75% 적용 |
| 소음 배관 1회 결과 | `+15` | `20 * 0.75` |
| 변이 선택 후 복귀값 | `25%` | 내부 대응 후 숙주 모드에 남는 잔여 경계 |
| 백혈구 접촉 포착값 | `+8` | 성공 복귀 시 변이 선택 후 복귀값에 가산 |
| 내부 미니게임 실패 후 복귀값 | `60%` | 실패 후 보상 없이 숙주 모드에 남는 높은 잔여 경계 |
| 면역 대응 경험 | 설계 반영, 구현 전 | 내부 미니게임 진입/성공/실패에 따라 다음 대응 강화 |

`시간 경과 또는 위험 행동으로 상승`이라는 상위 기획은 유지하되, 1차 플레이어블에서는 `위험 행동으로 상승`을 먼저 명확히 검증한다. 시간 경과 상승은 이후 HUD와 원인 피드백이 충분히 준비된 뒤 다시 켤 수 있다.

## 권장 경계 단계

초기 수치 기준은 다음처럼 읽히면 좋다.

| 구간 | 상태명 | 플레이어 의미 | 피드백 방향 |
| --- | --- | --- | --- |
| 0-24 | 잠복 | 아직 조용함 | 낮은 채도, 작은 게이지 |
| 25-49 | 국소 의심 | 숙주 내부가 이상을 감지함 | 게이지 강조, 짧은 경고음 |
| 50-74 | 경보 | 면역 압박이 체감됨 | HUD 색 변화, 위험 문구 |
| 75-99 | 동원 직전 | 내부 미니게임 전환이 가까움 | 맥박/진동성 피드백 |
| 100 | 면역 동원 | 내부 바이러스 모드 진입 | 전환 연출 |

## 설계 원칙

- 경계도 상승은 플레이어가 방금 한 행동과 연결되어야 한다.
- 원인 없는 자동 상승은 초반 프로토타입에서 최소화한다.
- 위험 행동은 손해만 주지 말고 정보, 지름길, 변이 조각, 진행 단서 같은 보상 후보와 연결한다.
- 오염 구역은 `배경 분위기`가 아니라 접촉, 체류, 상처 상태 같은 명확한 조건이 있을 때 경계도를 올린다.
- 숙주 조종은 항상 벌을 주지 않고, 숙주 본능을 거스르는 강제 조작에만 조종 부하를 준다.
- 경계도는 바이러스의 실패 상태가 아니라 `변이를 강제하는 압박`이다.
- 내부 미니게임은 경계도의 벌칙이 아니라 바이러스가 숙주의 방어에 적응하는 장면이어야 하며, 백혈구 전투 하나로 고정하지 않는다.
- 내부 미니게임 보상은 공짜 파밍이 아니어야 한다. 진입과 실패는 면역계의 대응 경험을 남기고, 실패는 바이러스 피해와 보상 제한으로 연결한다.
- 실제 면역학 용어는 분위기와 논리의 근거로만 사용하고, 치료나 의학적 정확성을 주장하지 않는다.

## 변이와의 연결

| 변이 계열 | 경계도와의 관계 |
| --- | --- |
| 잠복 강화 | 위험 행동 또는 시간 경과 상승량을 줄인다. |
| 신경 조종 | 강한 상호작용을 가능하게 하지만, 일부 행동은 경계도 상승량이 커질 수 있다. |
| 포유류 적응 | 쥐 이후 숙주 확장 후보를 열며, 포유류 숙주의 더 복잡한 면역 압박을 감당하는 방향으로 해석한다. |
| 공존 계열 | 숙주 손상과 면역 경계도 상승을 낮춰 장기 체류를 돕는다. |
